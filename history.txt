VSFilter (MPC-BE project)

2.47
----------------------
The VSFilter project is separated from the MPC-BE project.
Replace the <br> tag with a newline in SubRip subtitles.
Added support for WebVTT.
Fixed filter operation when changing the playback speed.
Fixed various subtitle display issues.
Fixed player crash on some corrupted subtitles.
Improved support for PGS subtitles.
Improved and optimized work with text in different encodings.
Fixed registration of a filter from a folder with Unicode characters.
For VirtualDub and AviSynth plugins, the DefaultCodePage parameter is now used instead of the CharSet parameter.
Fixed support for Unicode paths for the AviSynth plugin.
Fixed display of VirtualDub plugin settings.
Removed support for Avisynth v1.0.
Fixed saving subtitle search paths in the settings dialog.
Added the ability to use the '*' symbol for the last folder in the subtitle search path.
Removed support for RGB555 and RGB565 video.
Added support for AYUV format.
Fixed work with top-down and bottom-up bitmaps.
The "Flip picture vertically" and "Flip subtitles vertically" settings have been removed.
Added support for Rec.709 (will be used by default if Rec.601 is not recognized).
